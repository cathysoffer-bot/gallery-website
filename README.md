# ArtVista Gallery - Homepage

A modern, responsive homepage for an art gallery website featuring contemporary artwork and exhibitions.

## Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Modern UI**: Beautiful gradient designs and smooth animations
- **Interactive Elements**: Smooth scrolling navigation and hover effects
- **Gallery Showcase**: Featured artwork collection with overlay information
- **About Section**: Gallery information with highlighted features
- **Contact Information**: Location, hours, and contact details

## Files

- `index.html` - Main homepage structure
- `styles.css` - Styling and responsive design
- `script.js` - Interactive functionality

## How to View

Simply open `index.html` in a web browser, or serve it using a local web server:

```bash
python3 -m http.server 8080
```

Then navigate to `http://localhost:8080/index.html`

## Sections

1. **Navigation Bar**: Fixed header with links to all sections
2. **Hero Section**: Eye-catching introduction with call-to-action
3. **Gallery Section**: Grid of featured artwork with hover overlays
4. **About Section**: Gallery mission and key features
5. **Contact Section**: Visitor information and contact details
6. **Footer**: Copyright and social media links

## Browser Compatibility

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
